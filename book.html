Ruby 是基础，接触 Rails 前请务必先了解 Ruby。

Ruby

  • 《Programming Ruby》
    著名的搞头书，推荐先看这本，内容活泼易懂。前半部分涵盖了 Ruby 的方方面面，后半部分是函数库参考，所以不要被它的厚度所吓到。

  • 《Ruby 编程语言》
    本书的作者之一是松本行弘，Ruby 的缔造者。Ruby 开发者的必看书籍，被称为 Ruby 版的 K&R（K&R 指《The C Programming Language》，中文名《C 程序设计语言》，计算机史上的经典著作，其赞誉相当之高）。

  • 《Ruby 元编程》

  • 《Metaprogramming Ruby》

Rails

  • 《应用 Rails 进行敏捷 Web 开发》
    中英文均已更新至第四版。
    中文版是使用 Rails 3.0 ，英文版是 Rails 4.0 ，书中部分内容已经重写。英语好的还是建议阅读此书 Rails 4.0 英文版。

  • 《Ruby on Rails 实战圣经》
    在线中文资料，作者@ihower，博客。

  • 《Ruby on Rails Tutorial》
    在线中文版，翻译：@andor_chen

  • 《Objects on Rails》
